# stt-cli

Studio 3T CLI and MCP server for MongoDB.

## Install

**macOS / Linux**
```sh
curl -sSf https://raw.githubusercontent.com/3tio/3t-mcp-releases/main/install.sh | sh
```

**Windows (PowerShell)**
```powershell
irm https://raw.githubusercontent.com/3tio/3t-mcp-releases/main/install.ps1 | iex
```

## Update

```sh
stt-cli update
```
