v0.2.0 — Initial Release

stt-cli is the Studio 3T CLI and MCP server for Document Databases (Mongodb compatible)

MCP Tools

list_databases / list_collections / list_indexes — explore your MongoDB instance
query — execute queries with pagination and byte-budget controls
explain_query — get query execution plans
analyze_schema — infer schema from a collection sample
aggregate — run aggregation pipelines with validation and write-stage controls
scan_pii — detect PII fields in a collection
CLI

All tools available as stt-cli subcommands. Manage connections via stt-cli connections.

Auth

Browser-based login via stt-cli login. All query commands require authentication.

Install

curl -sSf https://raw.githubusercontent.com/3tio/3t-mcp-releases/main/install.sh | sh

Update

stt-cli update
