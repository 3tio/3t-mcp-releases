First public release of **3t-mcp** — a Model Context Protocol server and CLI for MongoDB, built by the Studio 3T team.

  ### MCP tools

  | Tool | What it does |
  |---|---|
  | `list_databases` | List databases with size and collection stats |
  | `list_collections` | List collections with storage and document stats |
  | `list_indexes` | List indexes for a collection |
  | `execute_query` | Run a MongoDB find query; accepts pipeline via stdin |
  | `explain_query` | Explain query execution plan |
  | `analyze_schema` | Sample documents and infer collection schema |
  | `aggregate` | Run an aggregation pipeline with pagination and response size limits |
  | `scan_pii` | Scan a collection for personally identifiable information |

  ### CLI

  All tools are available as `stt-cli` subcommands (`databases`, `collections`, `indexes`, `query execute`, `explain`, `schema analyze`, `aggregate`, `scan-pii`). Multi-connection support via YAML config.
